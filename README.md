# Pizza Store
it is pizza store to show the new pizzas and make orders via website


## User's Features 
- Login and Register
- Add to cart and make a order
- pay the order
## Admin's Features 
- View all (users , pizzas ,orders)
- Add pizza
- Update pizza
- delete pizza or user
- deliver order

## Technologies
- express js
- Redux
- React 
- mongodb
-stripe (payment tool)
